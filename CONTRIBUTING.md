## Contribuciones de Contenido

Si deseas contribuir al proyecto, puedes hacerlo a través de una **Pull Request**. Siguiendo estos pasos:

1. Crear la Pull Request
2. Agregar la siguiente información:
    - **Título**: Un título descriptivo para el artículo o recurso que estás agregando.
    - **Descripción**: Una descripción para el artículo.
    - **Contenido**: Aquí agregarás el contenido usando el markdown normal, ya que los archivos también son justamente markdown.
3. Etiquetar la Pull Request con la etiqueta `contribution` para que sea fácilmente identificable.
4. Esperar a que el equipo revise tu contribución y, si es aprobada, se fusionará con el proyecto!! ❤️

## Contribuciones de Código

Si deseas contribuir al código del proyecto, puedes abrir una pull request con tus cambios. De ser posible, incluye una descripción de los cambios realizados, qué arreglaste y demás detalles. Yo me encargaré de revisarlo manualmente!
